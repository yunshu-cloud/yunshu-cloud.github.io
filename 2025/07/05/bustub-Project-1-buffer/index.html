LRU replacer

  • Background
    The Least recently used cache is a cache replacement policy used in computing to manage how data is stored in the cache.
    When the cache is full and new data need to be stored, a decision must be madde about which old data to remove.This is where the LRU algorithm comes into play.
  • Thought
    The LRU cache policy works on the principle that data items that have not been used recently are likey to be used less in the future.Therefore, when the cache is full and a new item needs to be added, the LRU algorithm removes the item that has been used least recently.
  • Implementation
    To implement this policy, the cache keeps track of the order in which items are accessed. This can be done using a variety of data structures, such as linked lists or arrays.
